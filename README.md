# TaskPulse - 智能桌面专注助手 🍅

**TaskPulse** 是一款专为开发者和高效工作者设计的桌面生产力工具。它将**番茄工作法**、**任务追踪**与**系统提醒**完美融合，帮助您在繁忙的工作中保持专注，掌控时间。

---

## ✨ 核心功能 (Key Features)

### 1. 🍅 沉浸式专注模式
- **一键番茄钟**：首页醒目的红色按钮，一键开启 25 分钟标准番茄专注。
- **灵活时间设定**：提供 15 / 30 / 45 / 60 分钟常用快捷键，也可通过**更精准的滑动条**（0-120分钟）自由设定专注时长。
- **智能循环逻辑**：内置番茄工作法逻辑，完成 4 个番茄钟后自动提示长休息，劳逸结合。

### 2. 📋 可视化任务管理
- **清晰的任务面板**：采用 **表格化布局 (Grid Layout)**，清晰展示任务名称、起止时间及剩余倒计时。
- **视觉增强**：
  - **交替行底色**（斑马线），阅读不费力。
  - **列宽自动均分**，界面整洁平衡。
  - **完整信息展示**：鼠标悬停即可查看超长任务名称的全貌 (Tooltip)。
- **状态追踪**：进行中任务显示“⏳”，已完成任务自动标记为“✅”并变绿，成就感满满。

### 3. 🖥️ 极简与后台常驻
- **托盘运行**：点击关闭按钮自动**最小化到系统托盘**，不占用任务栏空间，后台静默守护。
- **原生通知**：支持 Windows 原生系统气泡通知，任务开始、结束、取消时即时提醒。
- **小巧精致**：优化后的紧凑窗口尺寸 (380x500)，圆角按钮设计，现代感十足。

### 4. ⚙️ 个性化设置
- **开机自启动**：一键设置，开机即用。
- **工程师模式**：(实验性功能) 可开启 Python 刷题提醒。
- **DeepSeek API 集成**：(开发中) 预留 AI 接口，未来将支持更强大的智能辅助功能。

## 🚀 快速开始 (Quick Start)

### 运行环境
- Windows 10/11
- Python 3.8+

### 安装步骤
1. **安装依赖**：
   在项目根目录下打开终端，运行：
   ```bash
   pip install -r requirements.txt
   ```

2. **启动应用**：
   - **方式一（推荐）**：直接双击运行根目录下的 `Start_TaskPulse.bat`。
   - **方式二（命令行）**：
     ```bash
     python -m src.main
     ```
